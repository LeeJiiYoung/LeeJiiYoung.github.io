선별된 프로젝트

가장 큰 성과를 이뤄낸 대표적인 사례 연구입니다.

WhaleOrder

주문 시스템과 장바구니를 직접 설계하고 멱등성, 분산락을 직접 구현해보고 싶었으며 AWS에 배포하는 경험이 필요했습니다.

  • 문제: 카페 주문 시스템의 전체 플로우(회원 → 메뉴 → 주문)를 직접 설계하고 운영 환경까지 구축하는 경험이 필요했습니다.
  • 해결: Java 21 + Spring Boot 3.5 기반으로 회원관리, 메뉴 등록, 주문 처리 기능을 구현하고, Redis를 활용한 장바구니 임시 저장 구조를 설계하여 DB 부하를 줄이고 빠른 응답속도를 확보했습니다. Docker 컨테이너화 후 AWS에 직접 배포하여 실 운영 환경을 구성했으며, QueryDSL·JPA 기반의 타입 안전한 데이터 계층을 설계했습니다.
  • 기술: Java 21, Spring Boot 3.5, PostgreSQL, Redis, QueryDSL, JPA, Docker, AWS, Git, Claude

국회사무처 인적자원 관리시스템 고도화 및 웹 표준 전환

기존 인적자원 관리시스템 고도화 및 추가 개발

  • 문제: PowerBuilder 기반의 노후 클라이언트 애플리케이션과 JSP 스크립틀릿으로 파편화된 레거시 시스템이 유지보수 한계에 도달하여 전면 현대화가 필요했습니다.
  • 해결: PowerBuilder 소스를 Java로 이관하고, JSP 스크립틀릿에 집중된 비즈니스 로직을 Java Service Layer로 분리하여 계층 구조를 정립했습니다. 화면은 WebSquare 기반 웹 표준으로 전환하였으며, DB 스키마 재설계 및 정규화를 통해 데이터 정합성을 확보했습니다. 화면 설계, 백엔드 개발까지 전 과정에 참여했습니다.
  • 기술: Java 17, Spring, eGovFramework, WebSquare5, Oracle, MyBatis, SVN
